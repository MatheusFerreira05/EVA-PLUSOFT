# O EVA é um chatbot inovador desenvolvido para ajudar os usuários a superar a paralisia de escolha em plataformas de streaming, como Netflix, Amazon Prime Video e HBO. Utilizando inteligência artificial, o EVA faz perguntas aos usuários sobre suas preferências de entretenimento e oferece recomendações personalizadas de filmes e séries. Com uma interface amigável e interativa, o EVA simplifica o processo de busca por conteúdo, tornando a experiência de descoberta mais eficiente e gratificante.

# Funcionalidades:

# Interatividade Conversacional: Simula uma conversa real entre o usuário e o assistente virtual.
# Personalização Sob Medida: Recomendações precisas com base nas preferências individuais de cada usuário.
# Experiência Amigável e Intuitiva: Interface intuitiva que simplifica a busca por entretenimento.
# Integração com Plataformas de Streaming: Acesso direto aos títulos recomendados nas principais plataformas de streaming.
